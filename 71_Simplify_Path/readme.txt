71_Simplify_Path


题目大意
Given an absolute path for a file (Unix-style), simplify it.

解决方案
首先要明白一些特殊的情形：
1. /../就是/。所以遇到..的时候不一定就是进入上一层，如果是在根目录那就是本层
2. ..a ... 等可以作为文件名
明白这两点之后就知道了，应该处理两个/之间的字符串，可能是:
/ . .. *
空 同 上 文件


one-time-attention
代码里我的做法是先删除开头的/，然后找到下一个/之前的字符，如果没有下一个/， 那就是剩余所有字符。但是所有字符的处理都是在while里的，所以/...这样子的就会错，应该要在最后加上/，确保所有需要处理的字符串后面都是跟着/的



样例
For example,
path = "/home/", => "/home"
path = "/a/./b/../../c/", => "/c"
