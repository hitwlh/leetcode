41_First_Missing_Positive


题目大意
Given an unsorted integer array, find the first missing positive integer.

解决方案
这题依赖两个观察：
1.由n个数构成的序列，返回的数字至多是n+1
因此如果没有空间是常数的要求的话，可以直接开一个bool的数组

2.可以令每个元素出现在它该出现的位置上，如1出现在num[0],2出现在num[1],3出现在num[2]...
这时有一个特性：如果一个元素，比如100出现在num[1]（即num[1] == 100）,那么如果num[99] != 100，就可以直接将num[1]和num[99]对调
这样100就出现在了它应该出现的位置，如果num[99] == 100，那么100本来就在正确的位置上，可以不用对调。
顺序遍历一次，这样所有元素都会出现在它该出现的位置上，而某个位置没有出现该元素就意味着这个元素不在数组里



样例
Given [1,2,0] return 3,
and [3,4,-1,1] return 2.
